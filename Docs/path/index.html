Path

Path (contains curves, lines and other). Parameters: points, [fill, [stroke]].

3 different ways:

var points = "M10,10 L200,200";

var points = [
    [10,10], // first is moveTo always
    [200,200], // 2 arguments - lineTo
    [350,250, 400,200], // 4 - quadraticCurveTo (hx, hy, x, y)
    [400,170, 70,10, 10,10] // 6 - bezierCurveTo (h1x, h1y, h2x, h2y, x, y)
];

var points = [
	{ name:'moveTo', arguments:[10,10] },
	{ name:'lineTo', arguments:[200,200] },
	{ name:'quadraticCurveTo', arguments:[350,250, 400,200] },
	{ name:'bezierCurveTo', arguments:[400,170, 70,10, 10,10] }
];

SVG-like

It's not SVG! But supports full SVG syntax.

  • M x,y - moveTo
  • L x,y - lineTo
  • Q hx,hy,x,y - quadraticCurveTo
  • C h1x,h1y,h2x,h2y,x,y - bezierCurveTo
  • Z - closePath

Functions

All canvas functions (moveTo, lineTo, quadraticCurveTo, bezierCurveTo, arc, arcTo, closePath).

points ( [points : string / array] )

Returns or changes path:

var path = ctx.path("M10,10 L200,200");
path.points();
/* -> [
    {name:'moveTo', _arguments:[10,10], x, y, set... },
    {name:'lineTo', ...}
] */

path.points("M100,100 L200,200");

point ( index : int, [ value : string / array ] )

Returns or changes one point (curve function).

var point = path.point(0);
point.name; // -> 'moveTo'
point.x; // -> 10
point.y; // -> 10
point.arguments(); // -> [10,10]

point.arguments(20,20); // redraws the path and updates x,y variables
point._arguments; // == point.arguments()
point.set('y', 20); // change argument by his name

We can replace any point to others.

path.point(1, 'L100,0');
// replace the second point to lineTo(100,0)

path.point(0, 'M0,0 L200,200');
// replace one point to two

before ( index : int, [ value : string / array ] )

Inserts the points before point with certain index.

var p = ctx.path('M10,10 L0,0');
p.before(1, 'L0,10');
// p = M10,10 L0,10 L0,0

after ( index : int, [ value : string / array ] )

remove ( [ index ] )

With index - removes the point. Without - removes the path.

Drawing curves

  • moveTo(x, y)
  • lineTo(x, y)
  • quadraticCurveTo(hx, hy, x, y)
  • bezierCurveTo(h1x, h1y, h2x, h2y, x, y)
  • arcTo(x1, y1, x2, y2, radius, clockwise)
  • arc(x, y, radius, start, end, clockwise)
  • closePath()

add ( name : string, arguments : array )

Inserts new curve to the end of path.

path.add('lineTo', [ 200,200 ]);

push ( curve : object )

Custom curve.

path.push({
    process : function(ctx){
        ctx.lineTo(20, 30);
        ctx.lineTo(30, 30);
    }
});

allPoints ( [function] )

Returns all points (bezier curves has 2 or 3 points - control points and end points).

// path - M10,10 L200,200 Q300,250,400,200

path.allPoints();
/* -> [
    [10,10], // moveTo
    [200,200], // lineTo
    [300,350], // quadratic handle
    [400,200] // quadratic point
]
*/

// with function -- works like Array::map
path.allPoints(function(x, y){
    return [x + 10, y + 10];
    // will translate path the to [10,10]
});

transformPath ( function : string, parameters : attributes, pivot )

Transforms path with changing coordinates of each point:

var path = ctx.path("M10,10 L200,200");

path.point(0); // moveTo 10,10

path.rotate(90);
path.point(0); // moveTo 10,10

// path rotated, but coordinates hasn't changed

path.transformPath('rotate', 90);
path.point(0); // moveTo 200,10
  • 'translate', x, y
  • 'scale', x, y, pivot
  • 'scale', size, pivot
  • 'rotate', angle, pivot
  • 'skew', x, y, pivot
  • 'skew', size, pivot
  • m11, m21, m12, m22, m13, m23, pivot
