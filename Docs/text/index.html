Text

Text. Just a text. Without line breaks. Parameters: text, [font], x, y, [fill, [stroke]].

ctx.text("Hello, world!", 10, 10, "black");
ctx.text("Bye, cruel JS", "Arial 15px", 10, 100, "red");

ctx.text({
    text : "Hey again, JS",
    x : 10,
    y : 150,

    fill : '#ffa',
    align : 'center'
});

Additional parameters: font, align, baseline, width (max width).

text, x, y

Return or change certain parameter.

tex1.text(); // -> "Hello, world!"
tex1.text("Again, cruel JS");
tex1.text(); // -> "Again, cruel JS"

font ( value : string / object )

tex1.font(); // -> "italic bold 10px Arial"
tex1.font(true); // -> { italic:true, bold:true, size:10, family:"Arial" }

tex1.font("20pt"); // changes only size
tex1.font({ bold:false });

// Graphics2D understands even so:
tex1.font('Comic 2px italic Sans bold MS');
tex1.font(); // -> 'italic bold 2px Comic Sans MS'

family / size / bold / italic

tex1.family("Myfont bold")
tex1.family(); // -> "Myfont bold"

tex1.size(); // -> 10
tex1.size('20px');

tex1.bold(); // -> true
tex1.bold(false);

align / baseline

Align : left, center, right.

Baseline (vertical align) : top, middle, bottom, ideographic, alphabetic.

underline ( enabled : boolean )

true or false.

width ( value : int )

Returns current width; changes maximum width.

tex1.width(); // -> 55.01953125
tex1.width(30);
tex1.width(); // -> 30
