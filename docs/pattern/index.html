Graphics2D API Documentation

Download Graphics2D.js and paste to your page:

<script src="Graphics2D.js"></script>

Well, now add canvas:

<canvas id="mycanvas" width="700" height="500"></canvas>

Let's get Graphics2D context:

var ctx = Graphics2D.id('mycanvas');

We can draw!

ctx.rect(10, 10, 200, 200, {
	colors:['#0af', '#fa0'],
	from:'top left',
	to:'bottom'
}, 'black 2px #f0a');
