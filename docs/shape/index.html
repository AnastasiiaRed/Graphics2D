Shape

Shape is a base class - all shapes inherits from him.

We can create any object in two ways:

// parameters
ctx.rect(10, 10, 200, 200, 'black', '2px blue');

// or object
ctx.rect({
    x: 10,
    y: 10,
    width: 200,
    height: 200,
    fill: 'black',
    stroke: '2px blue',

    opacity: 0.5 // additional property
});

Available properties: opacity, composite, visible, clip.

Also we can create hidden objects - they will respond to events.

fill ( color / gradient / texture : string / object )

Returns object fill, or changes it.

object.fill(); // -> 'red'
object.fill('blue');

Gradient - object with colors property. More is here.

object.fill({ colors:['red', 'green'], from:'top', to:'bottom' });
object.fill(); // -> [object Object]

Pattern - object with image property or string starts from ., http://.

stroke ( stroke:string / object / null )

Returns object stroke, or changes it.

object.stroke(); // -> Object(color, width, cap, join, dash)

object.stroke('2px red');
object.stroke('15px'); // changes only width; color is still 'red'
object.stroke({ cap:true, color:'blue' });

object.stroke(null); // removes stroke

Possible parameters:

  • width -- any css-like unit (5px, 0.5em). Without units - works as px.
  • color.
  • join -- miter, bevel, round.
  • cap -- butt, square, round. And: 2px round red -- cap = join = 'round'.
  • dash -- array ([2,2,1]) or shortdash, shortdot, shortdashdot, shortdashdotdot, dot, dash, longdash, dashdot, longdashdot, longdashdotdot.
  • opacity -- only float, only in text format. We can use green 0.5 without rgba(0, 128, 0, 0.5).

opacity ( value:float {from 0 to 1} )

Returns or changes opacity.

composite ( value:string )

All possible values is here: http://dev.w3.org/fxtf/compositing-1/.

show () / hide ()

Makes object hidden / visible.

cursor ( value:string )

Defines cursor on mouseover to object. All possible values is here: http://www.w3.org/TR/css3-ui/#cursor.

object.cursor('crosshair');

clip ( shape )

Returns or changes the object's clip.

object.clip(); // -> undefined

// 4 parameters - rect
object.clip( 10, 10, 200, 200 );

// 3 parameters - circle
object.clip( 100, 100, 50 );

// has the processPath function:
object.clip({
    processPath : function(ctx){
        ctx.beginPath();
        ctx.moveTo(0,0);
        ctx.lineTo(100,0);
        ctx.lineTo(50,100);
        ctx.closePath();
    }
});

// or:
object.clip( ctx.rect(10, 10, 200, 200) );

// other - path
object.clip("M10,10 L200,200");

// delete
object.clip(null);

We can change the clip:

object.clip( 100, 100, 100, 100 );
object.clip().animate('rotate', 45);

z ( value:int )

Returns or changes z-index of object.

Z-index is index of object in all objects array. Each object has his unique z-index; object with larger z-index will be drawn over objects with less z-index.

var a = ctx.rect(10, 10, 200, 200);
var b = ctx.circle(100, 100, 50);
a.z(); // -> 0
b.z(); // -> 1

b.z(0);
b.z(); // -> 0
a.z(); // -> 1

isPointIn ( x:int, y:int )

Returns true if point (x, y) is in the shape.

var rect = ctx.rect(10, 10, 10, 10);
rect.isPointIn(0, 0); // -> false
rect.isPointIn(11, 11); // -> true

bounds ()

Isn't defined in Shape class, but all Graphics2D shape classes contains it. Returns bounds of object.

ctx.circle(100, 100, 15)
    .bounds(); // -> { x:85, y:85, width:30, height:30, cx... }

Object has properties: x, y, width, height, w, h (short aliases for width & height), x1, y1, x2, y2 (left top and right bottom point), cx, cy (center coords).

remove ()

Removes the object.

Events

on ( event:string, function:function )

Adds listener to element. Supports all canvas events. Extends event object by 3 properties (targetObject, contextX, contextY). This == object.

object.on('mouseover', function(e){
    e.targetObject == this; // true
    e.contextX; // mouse coordinates on canvas
    e.contextY;
});

We can use quick calls:

object.on('click', 'fill', 'red');
// instead of
object.on('click', function(){
	this.fill('red');
});

And timers:

object.on(1000, function(){});
// instead of
setTimeout(function(){}.bind(object), 1000);

// and:
object.on(1000, 'fill', 'blue');

once ( event:string, function:function )

Listener, that will removed after first calling.

off ([event:string, [function:function]])

Removes 1) all listeners; 2) all listeners to certain event; 3) certain listener.

object.off('click', anyfunc);
object.off('click');
object.off();

fire ( event:string, [object:object] )

Calls all listeners of the certain event.

object.on('click', function(e){
	alert(e.which);
});
object.fire('click', { which:23 });

Aliases

Aliases for .on and .fire. The next events: click, dblclick, mousedown, mouseup, mousemove, mouseover, mouseout, mousewheel, focus, blur. For example:

object.click(function(e){
	alert(2);
});
// == object.on('click', function...)

object.mouseover('fill', 'red');

object.click();
// == object.fire('click');

Transformations

We can use the pivot parameter with all transform functions (except for translate).

Pivot is an array or object with absolute coordinates ([10, 10], { x: 15, y: 15 }) or string with relative coordinates (left (left center); top right == right top == tr == rt and other).

Pivot = center of shape by default.

scale ( size : float / object / array, [pivot] );     scale ( x : float, y : float, [pivot] )

Scales the shape.

object.scale(2);
object.scale(2, 'left top');

object.scale(2, 3);
object.scale(2, 3, 'right bottom');

object.scale([1, 0.5], 'center');
object.scale({ x:1, y:0.5 });

rotate ( angle, [pivot] )

Rotates the shape (using degrees).

object.rotate(45);
object.rotate(-45, 'top');

skew ( size : float / object / array, [pivot] );     skew ( x : float, y : float, [pivot] )

Skew transform; using degrees.

object.skew(10);
object.skew(-10, 'bottom');

object.skew(5, 0);
object.skew(0, 5, 'rb');

object.skew([-5, -5], 'lt');
object.skew({ x:0, y:0 });

translate ( x : int, y : int )

Translates shape.

object.translate(10, 10);

transform ( m11, m21, m12, m22, m13, m23, [pivot] )

Transforms by matrix.

object.transform(
    Math.cos(Math.PI / 4),
    Math.sin(Math.PI / 4),
   -Math.sin(Math.PI / 4),
    Math.cos(Math.PI / 4),
    0,
    0,

    'center'
);
// will rotate object to 45 degrees (Pi / 4 radians)

More about matrix transformations: http://en.wikipedia.org/wiki/Transformation_matrix.

Animation

animate ( parameter : string, value, [duration, [easing, [callback]]] )

animate ( parameter : string, value, [options : object] )

animate ( object : object, [duration, [easing, [callback]]] )

animate ( object : object, [options : object] )

Duration - int; easing - string / function; callback - function.

object.animate('width', 100);
object.animate('height', 300,
   1000, 'bounceOut', function(){ this.hide() });

object.animate({
    x : 10,
    fill : 'red',
    stroke : '3px'
}, {
    duration : 5000, // ms
    easing : function(x){ return Math.sin(x * Math.PI); }
});

Available parameters:

  • opacity.
  • fill (except for gradients & patterns).
  • stroke (only width & color).
  • crop (for images).
  • x, y, width, height (rect).
  • cx, cy, radius (circle).
  • rotate, scale, scaleX, scaleY, skew, skewX, skewY, translate, translateX, translateY (transformations; pivot = 'center').
