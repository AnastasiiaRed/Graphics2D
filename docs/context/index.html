Context

Basic Graphics2D context. We can use on of this functions:

var ctx = Graphics2D.id('mycanvas');
var ctx = Graphics2D.query('canvas'); // the first canvas
var ctx = Graphics2D.query('canvas', 1); // the second canvas
var ctx = Graphics2D.query( document.getElementById('mycanvas') );

getObjectInPoint ( x, y )

Returns object in (x,y).

var first = ctx.rect(0, 0, 10, 10, 'red'),
    second = ctx.rect(5, 5, 10, 10, 'blue');
ctx.getObjectInPoint(7, 7); // == second

push ( object )

Adds your object to redrawing (or event processing). More is here.

on( event, function )

Adds listener to canvas. Event object extended by 3 properties.

ctx.on('click', function(e){
	e.targetObject; // object or null
	e.contextX; // coordinates of mouse on context
	e.contextY;
});

once( event, function )

Listener, that will removed after first calling.

off( [event, [function]] )

Removes 1) all listeners; 2) all listeners to certain event; 3) certain listener.

ctx.off('click', anyfunc);
ctx.off('click');
ctx.off();

update ()

Redraws canvas in 1ms (without repeating):

ctx.update();
ctx.update();
ctx.update(); // 3 calls, 1 update

Use __update for instant update.

You don't need this function, all objects update canvas themselves.
