Gradient Pattern

Gradient

Gradient may be linear or radial.

Linears have two points - start and end.

Radials have a radius in each point.

// relative coordinates - from top to bottom
var grad1 = ctx.gradient('linear', ['green', 'blue'], 'top', 'bottom');

// absolute coordinates
var grad2 = ctx.gradient('radial',
    {"0": 'blue', "0.5": 'red' }, [100,100,0], [100,100,100]);

// special radial parameters
var grad3 = ctx.gradient({
    type: 'radial',
    colors: ['red', 'orange', 'yellow', 'green', 'cyan', 'blue', 'purple'],
    center: 'center',
    startRadius: 10,
    radius: 100
});

ctx.rect(0, 0, 10, 10, grad1);

Updating a gradient will update all shapes, that was filled him!

Inline gradients

ctx.rect(0, 0, 10, 10, {
    colors: ['red', 'transparent'],
    from: 'top',
    to: 'bottom',
    type: 'linear'
});

colorMix ( t : float )

Return the color:

ctx.gradient('linear', ['white', 'black'], 'top', 'bottom')
       .colorMix(0.5); // -> [128, 128, 128, 1]

color ( t : float, color )

Returns / changes a color.

// grad1: red -> blue

grad1.color(0); // 'red'
grad1.color(1); // 'blue'
grad1.color(0.5); // undefined

grad1.color(0.5, 'green');

// grad1: red -> green -> blue

Difference between color and colorMix: colorMix calculates an undefined color.

colors ( value : array / object )

Returns (as object {stop : color}) or changes all colors.

grad.colors(); // -> {0:'red', 0.5:'green', 1:'blue'}
grad.colors(['red', 'blue']);
grad.colors(); // -> {0:'red', 1:'blue'}

from / to

Returns / changes. We can use array with absolute coordinates or string with relative (top left, tl).

Radial

radius / startRadius ( radius / point )

If we're using point, radius will have a distance from the shape center to the point.

grad2.startRadius(); // -> 0
grad2.radius(); // -> 100

grad2.startRadius(10)
     .radius([100,100]) // point -- array
     .radius(100, 100); // point -- arguments

center / hilite ( point : array )

center is a center of the gradient, hilite - offset.

from / to

Of course, we can use from / to with radial gradients too.
