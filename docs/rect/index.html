Rect

Rectangle. Parameters: x, y, width, height, [fill, [stroke]].

ctx.rect(10, 10, 200, 200, 'black');

ctx.rect({
    x : 100,
    y : 100,
    width : 50,
    height : 50,

    stroke : 'green 3px'
});

x, y, width, height

Return or change.

rect.x(); // -> 100
rect.x(200);
    .y(300)
    .width(100);
rect.x(); // -> 200
rect.width(); // -> 100

x1, y1, x2, y2

Return or change coordinates of the left top or right bottom point.

// rect - 10, 10, 200, 200
// (x, y, width, height)

rect.x1(); // -> 10
rect.x2(); // -> 210

rect.x1(0);
// rect - 0, 10, 210, 200

rect.x2(100);
// rect - 0, 10, 110, 200
