Rect

Прямоугольник. Параметры: x, y (координаты левой верхней точки), width, height (ширина и высота). Как всегда, дальше есть заливка и обводка.

var rect = ctx.rect(10, 10, 200, 200, 'black', 'blue 4px');
var rec2 = ctx.rect({
  x: 20,
  y: 20,
  width: 300,
  height: 300
});

Методы

x, y, width, height

Обычные методы, регулирующие параметры: при вызове без аргумента - возвращают значение, с - изменяют.

rect.x(); // -> 10
rect.x(100);
rect.x(); // -> 100

x1, y1, x2, y2

Немного другой принцип: 1 точка - левый верхний угол прямоугольника, 2 - правый нижний. Мы можем получать их координаты и менять (причём: при изменении первой, координаты второй точки меняться не будут!)

var rect = ctx.rect(10, 10, 200, 200);
rect.x1(); // -> 10
rect.x2(); // -> 210

rect.x1(0);
rect.width(); // -> 210

rect.x2(50);
rect.width(); // -> 50
