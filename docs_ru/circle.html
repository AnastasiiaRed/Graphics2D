Circle

Круг. Параметры: cx, cy (координаты центра) и radius (радиус).

var circ = ctx.circle(200, 200, 50, null, '4pt round black');
// fill=null

var cir2 = ctx.circle({
	cx: 200,
	cy: 200,
	radius: 25
});

Методы

cx, cy, radius

Параметры: при вызове без аргумента - возвращают значение, с - изменяют.

circ.cx(); // -> 200
circ.radius(); // -> 50
circ.radius(150);
circ.radius(); // -> 150
