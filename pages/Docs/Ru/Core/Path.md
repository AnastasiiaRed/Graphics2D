## Инициализация
```js
ctx.path(d, [offsetX, offsetY, fill, stroke]);

ctx.path({
	d, x, y, fill, stroke
});

Delta.path(...);
```

## Параметры
 - `d`
 - `x`
 - `y`

## Методы
 - `curve(index, [value])`
 - `before(index, [value, turnMoveToLine])`
 - `after(index, [value])`
 - `remove(index)`
 - `push(method, attrs)` or `push(curve)`
 - `each`
 - `map`
 - `moveTo`, etc
 - `transformPath`
