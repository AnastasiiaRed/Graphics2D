Textblock

Text block. We use block coordinates and can break text (with \n or auto break).

Parameters : text, [font], x, y, [width], [fill, [stroke]].

var tb, auto;
tb = ctx.textblock("Hello,\ntextblock!", "Arial 0.7em", 10, 10, 'black');
auto = ctx.textblock("It's must be a very long text, " + 
           "and it will auto break, because we're using " +
           "textblock", "Arial 13pt", 10, 10, 70, 'red');

Inherited from the Text class

x, y, font, family, size, bold, italic, align, underline. Baseline doesn't supported.

width ( value : int )

Returns / changes width of the block.

t = ctx.textblock("Lorem ipsum and etc...", "Arial 13pt", 10, 10, 'red');
// block without width

t.width(); // -> 171.6767578125

t.width(100); // autobreaks the text
t.width(); // -> 100

height ()

Returns the height.

t.height(); // -> 51

lineHeight ( value : int )

Returns / changes.

t.lineHeight(); // -> 17
t.lineHeight(30);
t.lineHeight(); // -> 30

limit ( value : int )

Lines limit. Returns / changes.
