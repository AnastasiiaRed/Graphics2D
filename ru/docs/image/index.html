Image

Image. Parameters: image, x, y, [width, height].

We can use 'native' (original image) or 'auto' (with proportions saving) width / height.

ctx.image("./image.jpg", 10, 10);

// document.getElementById('image');
ctx.image("#image", 50, 50, 200, 'auto');

var i = new Image;
i.src = 'http://keyten.github.io/Graphics2D/images/crossword.png';
ctx.image({
    image : i,
    x : 100,
    y : 100,

    crop : [0, 0, 50, 50] // we can use crop
});

x, y, width, height

Works like the same methods of Rect class. And width / height can be used with 'native' and 'auto'.

x1, y1, x2, y2

Works like the same methods of Rect class.

crop( value : array / arguments )

Returns or changes the crop:

image.crop(); // -> [0,0,10,10]
image.crop(10, 10, 200, 200);
image.crop([10, 10, 200, 200]);
